 Enable recurring payments for Moneris gateway in the GiveWP donation plugin.

== Description ==

This plugin requires the GiveWP core and GiveWP Recurring add-on plugins activated to function properly. When activated, it adds the ability to accept recurring (subscription) donations to the Moneris payment gateways.

== Installation ==

= Minimum Requirements =

* WordPress 5.5 or greater
* PHP version 7.4 or greater
* MySQL version 8.0 or greater
